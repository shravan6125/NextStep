# Career Guidance App  
🚀 An AI-powered career guidance application built with **React Native, Node.js, Express, and MongoDB**.  

## Features  
✔ AI-generated career roadmap  
✔ One-on-One mentorship (Realtime chat)  
✔ AI-powered resume score finder  
✔ AI chatbot integration  

## Tech Stack  
- Frontend: React Native  
- Backend: Node.js, Express.js  
- Database: MongoDB  

## How to Run  
```sh
git clone https://github.com/YOUR_GITHUB_USERNAME/career-guidance-app.git
cd career-guidance-app
npm install
npm start
